Carafe DataTable Download Builder CDN

DataTables suggests that you use the download builder to build a custom DataTables package including only the software you need. As this appears to be a simple solution for many users, we have a Carafe configurable package.

In the example below, the configurator was used to build a DataTable with the following options:

Styling Framework

  • Bootstrap 4

Packages

  • jQuery 3
  • DataTables

Extensions

  • Buttons (Column Visibility, HTML5 Export (Copy, Excel, PDF), Print View)
  • ColReorder (by dragging column headers)
  • FixedHeader (scroll down and header will stick to top)
  • Responsive (make the window small and a detail popup will be enabled)
  • RowGroup (grouped when sorted by Office column)
  • Select (click on rows to select / (opt/shift) click to extend/restrict selections)
